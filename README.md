## 關於頌缽療癒

頌缽（Singing Bowl）源自於西藏與喜馬拉雅山區的古老療癒工具，透過金屬缽的共鳴振動，幫助身體與心靈達到深層放鬆與平衡。

### 七脈輪頌缽清理

針對人體七大脈輪（海底輪、臍輪、太陽神經叢、心輪、喉輪、眉心輪、頂輪）進行音頻共振療癒，促進能量流動、釋放身心壓力。

### 頌缽美顏拉提

結合頌缽振動與臉部穴位，透過音波刺激促進臉部循環，達到自然提拉緊緻的效果。
